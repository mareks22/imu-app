# Robothub application with gyroscope

## To run the application:

- Create new app in robothub.
- Create folder structure like this:
 ```
 |-fronted
 |     |- *.glb //3d model
 |     |-index.html
 |     |-assetes //from dist folder
 |           |- *.js
 |           |- *.css
 |-app.py
 ```
 - Connect camera
 - Install robothub_depthai package
 - Start app
 - Navigate to app view
